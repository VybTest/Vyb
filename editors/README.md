# Editor Support

Syntax highlighting for Vyb test files.

## Editors

- [**VS Code**](vscode/) - TextMate grammar
- [**Vim/Neovim**](vim/) - Vim syntax file
- [**Sublime Text**](sublime-text/) - Sublime syntax

## Installation

See the README in each editor's folder for installation instructions.

## Contributing

To add support for another editor:

1. Create a folder with the editor name
2. Add the syntax/grammar file
3. Add a README with installation instructions
4. Submit a PR
